Sistema de Gestión de Usuarios y Automatización de Procesos

Python SQL PyQt JSON
Ver en nueva pestaña

Aplicación desarrollada en Python para la gestión de usuarios y automatización de procesos internos. El sistema trabaja conectado a una base de datos SQL y permite la administración de usuarios, generación de reportes y manejo de archivos locales, integrándose directamente con funcionalidades del sistema operativo.

Funcionalidades clave

  • CRUD de usuarios conectado a base de datos SQL
  • Visualización de información en interfaz gráfica
  • Consulta de bitácora de ingresos
  • Generación automática de reportes en PDF
  • Lectura y almacenamiento de configuraciones desde archivos JSON
  • Visualización de imágenes usando el visor por defecto de Windows

Qué demuestra este proyecto

Trabajo sobre sistemas existentes Manejo de persistencia Automatización real Integración con sistema operativo

Aplicación Móvil Android para Gestión Agrícola y Recolecciones

Ionic Angular TypeScript JWT APIs REST
Ver en nueva pestaña

Aplicación móvil desarrollada con Ionic y Angular para Android, orientada a la gestión de procesos agrícolas. La app se integra con APIs REST existentes, permite trabajo offline, autenticación segura y uso de hardware del dispositivo móvil.

Funcionalidades clave

  • Autenticación mediante JWT
  • Gestión de usuarios vía APIs
  • Registro de pesos de café y jornales
  • Consulta de recolecciones
  • Generación de PDF y Excel (Sharesheet)
  • Integración con cámara, Bluetooth y NFC
  • Almacenamiento local para uso offline
  • Sincronización posterior con el servidor

Qué demuestra este proyecto

Consumo de APIs reales Desarrollo móvil productivo Manejo de hardware Arquitectura offline-first
